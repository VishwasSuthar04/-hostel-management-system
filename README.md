# Hostel Management System

Hostel Management System is a full-stack web application developed to digitize and simplify hostel administration. The system helps manage student records, room allocation, fee management, mess details, and communication between hostel administration and students.

## Features

### Admin Panel
- Manage student records (add, update, delete)
- Room allocation and availability tracking
- Fee management and payment records
- Mess management system
- Search student records (Name, CNIC, Father Name, Entry Date)
- Send notifications to students
- View and respond to student complaints

### Student Panel
- Secure login system
- View personal profile and room details
- Check monthly fee status
- Upload payment proof
- Receive notifications from admin
- Submit complaints

### Landing Page
- Hostel introduction
- Room information with images
- Mess menu display
- Rules and regulations
- Contact form to communicate with admin
